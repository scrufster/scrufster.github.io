Download

PrimerPipeline runs on Windows XP and onwards.

DOWNLOAD PRIMERPIPELINE.ZIP

System Requirements

If you are using Windows 7 or later then you will be able to run PrimerPipeline.

If you are using Windows XP, then you must ensure that you have Microsoft .NET Framework 4.0 installed.

Note: To find out which version of the .NET Framework you have on your PC, go to:

Start > Control Panel > Add/Remove Programs

And look for Microsoft .NET Framework. If you don't have it, .NET Framework 4 can be downloaded from Microsoft here.

Installation

  1. Download PrimerPipeline.zip from the link below.
  2. Unzip the archive. It should contain the following files:

    • PrimerPipeline.exe

      This is the PrimerPipeline program itself.

    • PrimerPipeline_ Instruction_manual_v1.0

      The user guide for PrimerPipeline.

    • primer3_core.exe

      This is Primer3. PrimerPipeline will run this program in the background during the Primer3 stage of the process.

    • primer3_config

      This directory contains settings files used by Primer3.

    • example.fasta

      A test file.

  3. Place the entire extracted folder somewhere on your PC. It is not recommended to do this in the root of your C drive, or in Program files, - doing so will require you to run PrimerPipeline as an administrator in order for it to write files.
  4. If required, make a shortcut to PrimerPipeline.exe.
