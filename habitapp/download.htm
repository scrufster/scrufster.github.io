System Requirements - Windows

If you are using Windows 7 or later then you will be able to run HabitApp.

If you are using Windows XP, then you must ensure that you have Microsoft .NET Framework 4.0 installed.

Note: To find out which version of the .NET Framework you have on your PC, go to:

Start > Control Panel > Add/Remove Programs

And look for Microsoft .NET Framework. If you don't have it, .NET Framework 4 can be downloaded from Microsoft here.
